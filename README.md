## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/daniyal-abbassi/Rick-And-Morty-WiKi.git
    ```

2. Navigate to the project directory:
    ```bash
    cd Rick-And-Morty-WiKi
    ```

3. Install dependencies:
    ```bash
    npm install
    ```

4. Start the development server:
    ```bash
    npm run dev
    ```

## Project Structure
